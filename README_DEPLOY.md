# 心境探店 · MoodPlace — 评委运行指南

> 本指南面向评审专家，帮助您在 2 分钟内体验作品核心功能。

---

## ✅ 方式一：在线直接访问（推荐，30秒进入）

**体验链接**：https://moodplace-62m5.onrender.com

1. 用手机或电脑浏览器打开上方链接
2. 点击「开始探索」
3. 选择一种情绪（如"疲惫"），输入城市（如"武汉"）
4. 点击「为我寻找心仪之所」
5. 30 秒内查看推荐结果与氛围图

> 提示：Render 免费版有冷启动机制，首次访问可能需要 30-60 秒唤醒，之后响应正常。

---

## 💻 方式二：本地运行（备用，适合深度评审）

**前提**：已安装 Python 3.9+

**Windows**：
1. 解压代码包
2. 双击 `prototype/v4/run.bat`
3. 浏览器自动打开

**Mac / Linux**：
1. 解压代码包
2. 终端执行：`cd prototype/v4 && bash run.sh`
3. 浏览器自动打开

---

## 🔑 环境变量说明（如需自行配置 API Key）

| 变量名 | 说明 | 获取方式 |
|--------|------|---------|
| `VIVO_API_KEY` | vivo AI 开放平台 API 密钥 | vivo 开发者平台申请 |

设置方式：
- Windows：`set VIVO_API_KEY=你的密钥`
- Mac/Linux：`export VIVO_API_KEY='你的密钥'`

## 🎯 核心功能体验 checklist

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 选择情绪标签 + 输入城市 | 页面进入加载管道 |
